# CAR PRICE PREDICTOR APP
A Flask based web app. It predicts the probable price at which a certain car must be sold.

Frontend :

1. HTML, CSS

2. Bootstrap

3. JavaScript

Backend :

1. Python (Flask)

2. JavaScript

3. ML Model used = MLPRegressor
